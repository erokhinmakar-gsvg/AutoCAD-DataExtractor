AutoCAD Data Extractor
PowerShell GUI-скрипт для извлечения данных из DWG-файлов(генеральных планов):
суммарные длины линейных объектов по слоям
суммарные площади штриховок по слоям (площадные объекты)
длины дуг с группировкой по слоям и радиусу (R1…R12) (стандартные радиусы бортовых камней)
количество блоков (в т.ч. динамических с учётом состояния видимости)
Требования
Windows (10/11)
AutoCAD 2020, 2021 или 2022 (должен быть установлен локально)
PowerShell
Запуск
Скачайте или клонируйте репозиторий.
Запустите `start.bat` (или откройте PowerShell с правами пользователя и выполните `.\\DataExtractor.ps1`).
Выберите версию AutoCAD, укажите DWG-файл и нажмите нужную кнопку (Длины, Площади, Дуги, Блоки).
Результат появится во вкладках.
Экспорт в CSV или Excel (кнопки внизу).
Примечания
Скрипт работает через COM-объекты AutoCAD, поэтому AutoCAD должен быть закрыт при первом запуске (скрипт сам запустит скрытый экземпляр) или уже открыт.
Для расчёта дуг полилинии автоматически взрываются (оригинальный чертёж не сохраняется, изменения происходят только в памяти).
Штриховки с нулевой площадью выводят предупреждение.
При работе с блоками можно фильтровать по слою.
Затем dwg-файл закрывается без сохранения
